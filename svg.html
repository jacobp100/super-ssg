SVG

The InlineSvg component renders SVG inline on the page. It takes a src prop as defined in the assets structure section.

The <use> element is fully supported, and will rewrite the path to the /res folder.

import { InlineSvg } from "super-ssg";

<InlineSvg src="/path/to/your.svg" />;

<svg version="1.1" width="300" height="200" xmlns="http://www.w3.org/2000/svg">
  <rect width="100%" height="100%" fill="red" />
  <circle cx="150" cy="100" r="80" fill="green" />
</svg>
